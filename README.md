# KijaniKiosk DevOps Documentation

This repository contains engineering documentation, Git workflow guides, operational runbooks, and DevOps collaboration practices used by the KijaniKiosk engineering team.

The repository serves as a training environment for learning Git, branching strategies, pull requests, CI/CD pipelines, release management, and incident response procedures.
KijaniKiosk uses GitFlow and CI/CD pipelines for reliable releases.

## Milestone: week2-tuesday-ready

This milestone marks the completion of the Week 2 Tuesday DevOps exercises for the KijaniKiosk project.

Completed activities include:

* Repository initialization and Git history management
* GitFlow branch structure creation
* Feature branch development
* Pull request simulation and review workflow
* Merge conflict resolution
* Git recovery techniques
* CI workflow validation
* Release tagging and traceability practices

This tag serves as a checkpoint that allows the engineering team to identify the repository state at the completion of the Tuesday training activities.
